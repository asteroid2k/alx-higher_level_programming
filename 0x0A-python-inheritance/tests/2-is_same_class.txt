The ``2-is_same_class`` module
======================

Using ``is_same_class``
-------------------

Import the ``is_same_class`` class from the ``2-is_same_class.py`` module:

    >>> is_same_class = __import__("2-is_same_class").is_same_class

Test Cases:
    1. Same class
    >>> is_same_class(1,int)
    True

    1. Child class
    >>> class IntChild(int):
    ...     pass
    >>> is_same_class(1,IntChild)
    False
